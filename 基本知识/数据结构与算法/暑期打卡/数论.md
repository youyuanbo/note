# 数论

质数判定：

1. 试除法

   ```java
   public void isPrime(int n) {
       //默认n是质数
       boolean flag = true;
       //从2开始枚举，用i去除n，取余数
       for (int i = 2; i < n; i++) {
           //如果余数为0，说明不是质数
           if (n % i == 0) {
               //改变标志
               flag = false;
           }
       }
       //输出
       System.out.println(flag);
   }
   ```

2. 试除法升级

   ```java
   public void isPrime(int n) {
       //默认n是质数
       boolean flag = true;
       //从2开始枚举，用i的平方去除n，取余数
       for (int i = 2; i * i < n; i++) {
           //如果余数为0，说明不是质数
           if (n % i == 0) {
               //改变标志
               flag = false;
           }
       }
       //输出
       System.out.println(flag);
   }
   ```

查找1~n之间的所有质数：

1. 朴素筛法

   ```java
   //朴素筛选
   public void filter(int n) {
       int[] isPrime = new int[1000000];
       for (int i = 2; i <= n; i++) {
           if (isPrime[i] == 0) {
               System.out.println(i);
               for (int j = i + i; j <= n; j += i) {
                   isPrime[j] = 1;
               }
           }
       }
   }
   ```

2. 线性筛法

   ```java
   //线性筛法
   public void lineFilter(int n) {
   
       int[] isPrime = new int[n + 1];
       int[] primes = new int[n];
       int index = 0;
       for (int i = 2; i <= n; i++) {
           if (isPrime[i] == 0) {
               primes[index++] = i;
           }
           for (int j = 0; j < index && i * primes[j] <= n; j++) {
               isPrime[primes[j] * i] = 1;
               if (i % primes[j] == 0) {
                   break;
               }
           }
       }
   
       for (int i = 0; i < index; i++) {
           System.out.println(primes[i]);
       }
       System.out.println("index = " + index);
   }
   ```
