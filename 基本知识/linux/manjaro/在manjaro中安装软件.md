sudo pacman -S yay

中英文切换

```bash
#中文->英文
export LANG=en_US.UTF-8
xdg-user-dirs-update --force
#确认更新
#英文->中文
export LANG=zh_CN.UTF-8
xdg-user-dirs-update --force
#不更新，就完成切换

```
